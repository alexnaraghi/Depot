## v3.0 Requirements

### Resolve

- [x] **RSLV-01**: User sees conflict indicator on files needing resolution after sync
- [x] **RSLV-02**: User can launch external merge tool on conflicted files

### Depot Browser

- [x] **DPOT-01**: User can browse full depot hierarchy in a tree view
- [x] **DPOT-02**: User can sync files/folders from depot browser

### Workspace Management

- [x] **WKSP-01**: User can view list of available workspaces
- [x] **WKSP-02**: User can switch to a different workspace
- [x] **WKSP-03**: User can view client spec (read-only)

### Stream Management

- [x] **STRM-01**: User can switch to a different stream
- [x] **STRM-02**: Default CL files are auto-shelved when switching streams

### Search

- [x] **SRCH-01**: User can perform operations on search results via context menu
- [x] **SRCH-02**: User can click a submitted CL to view its details
- [x] **SRCH-03**: User can click an author to list their recent changes

### Auto-Refresh

- [x] **RFSH-01**: Workspace state auto-refreshes at a configurable interval
- [x] **RFSH-02**: User can manually refresh via button

### Settings

- [x] **STNG-01**: User can configure external editor
- [x] **STNG-02**: User can configure auto-refresh interval

### Testing

- [x] **TEST-01**: E2E test suite validates core workflows (sync, checkout, submit, revert)

### Bug Fixes

- [x] **BUGF-01**: Changelist drag-and-drop works reliably
- [x] **BUGF-02**: Editing default CL description moves files to new CL
- [x] **BUGF-03**: Unshelve places files in same changelist (not default)
- [x] **BUGF-04**: Resolve dialog appears after unshelving conflicting files
